PersonalityCamera
🛡️ 개인정보 보호 우선 정책
PersonalityCamera는 사용자의 개인정보 보호를 최우선으로 합니다. 모든 데이터는 기기 내부에서만 처리되며, 외부로 전송되지 않습니다.

1. 수집하는 개인정보

PersonalityCamera 앱은 다음과 같은 정보를 처리합니다:

  • 카메라 이미지: 성격 분석을 위한 사진 촬영
  • 분석 결과: AI가 생성한 성격 분석 데이터
로컬 저장 전용
서버 전송 없음
외부 공유 차단

2. 개인정보 처리 방법

📸 이미지 처리

  • 촬영된 모든 이미지는 기기 내부 저장소에만 저장됩니다
  • AI 분석은 기기 내부에서만 수행됩니다
  • 이미지나 분석 결과가 외부 서버로 전송되지 않습니다
  • 네트워크 연결 없이도 모든 기능이 정상 작동합니다

🤖 AI 분석 데이터

  • 성격 분석 결과는 기기 내부에만 저장됩니다
  • 분석 알고리즘은 오프라인으로 작동합니다
  • 개인 데이터의 외부 유출 위험이 없습니다

3. 앱 권한 사용 목적

⚠️ 필수 권한 안내
PersonalityCamera는 핵심 기능 수행을 위해 다음 권한을 요청합니다.

📷 카메라 권한

  • 사용 목적: 성격 분석을 위한 사진 촬영
  • 처리 방식: 실시간 카메라 화면 표시 및 이미지 캡처
  • 데이터 보관: 기기 내부 저장소에만 보관

💾 저장소 접근 권한

  • 사용 목적: 촬영한 이미지 및 분석 결과 저장
  • 처리 방식: 앱 전용 폴더에 안전하게 저장
  • 접근 범위: PersonalityCamera 앱 폴더로 제한

4. 개인정보 보호 조치

  • 오프라인 처리: 모든 데이터는 기기 내부에서만 처리
  • 암호화 저장: 민감한 데이터는 암호화되어 저장
  • 접근 제한: 다른 앱에서 데이터 접근 불가
  • 자동 삭제: 사용자가 앱을 삭제하면 모든 데이터 함께 삭제

5. 제3자 제공 및 위탁

🚫 제3자 제공 없음
PersonalityCamera는 수집된 개인정보를 제3자에게 제공하지 않습니다. 모든 데이터는 기기 내부에서만 처리되므로 외부 유출 위험이 없습니다.

6. 개인정보 보관 및 파기

📅 보관 기간

  • 이미지 및 분석 결과: 사용자가 직접 삭제할 때까지
  • 임시 처리 데이터: 앱 종료 시 자동 삭제

🗑️ 파기 방법

  • 개별 삭제: 앱 내 갤러리에서 개별 항목 삭제
  • 전체 삭제: 앱 삭제 시 모든 데이터 자동 파기
  • 설정 초기화: 앱 설정에서 모든 데이터 삭제 가능

7. 이용자의 권리

PersonalityCamera 사용자는 다음과 같은 권리를 가집니다:

  • 열람권: 저장된 이미지 및 분석 결과 확인
  • 수정권: 분석 결과에 대한 개인 메모 추가/수정
  • 삭제권: 언제든지 개별 또는 전체 데이터 삭제
  • 처리정지권: 앱 기능 사용 중단 및 데이터 처리 정지
